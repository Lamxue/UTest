

# 订单构成

## 主机 + 本地磁盘

系统盘本地磁盘情况下，主机的订单构成包括：

  - CPU
  - GPU（仅GPU机型）
  - 内存
  - 本地系统盘
  - 本地数据盘
  - 镜像（指镜像市场的收费镜像）
  - 附加特性（如网络增强）

外网弹性IP（EIP），外挂数据盘云硬盘（UDisk）不包含在内，是独立的订单，享有独立的生命周期。

## 主机 + 云硬盘

系统盘为云硬盘情况下，主机的订单构成包括：

  - CPU
  - 内存
  - 云系统盘
  - 镜像（指镜像市场的收费镜像）
  - 附加特性（如网络增强）

外网弹性IP（EIP），云数据盘（UDisk）不包含在内，是独立的订单，享有独立的生命周期。
